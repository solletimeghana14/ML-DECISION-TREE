Run following command for Random Forest
 
python Run_RandomForest.py train_data valid_data test_data
arguments:
train_data give the path of train dataset
valid_data give the path of valid dataset
test_data give the path of test dataset
                                      

Python Version Used is 3.7.3
Numpy,Pandas,Math functions are imported for all 6 algorithms(Run_DecisionTree_Classifier.py)
